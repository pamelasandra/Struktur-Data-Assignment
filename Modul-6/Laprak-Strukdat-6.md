#### A. Doubly Linked List (DLL)
Doubly Linked List merupakan pengembangan dari linked list di mana setiap elemen (node) memiliki keterhubungan dua arah. Berbeda dengan singly linked list yang hanya bisa bergerak maju, setiap elemen pada doubly linked list dibekali dengan dua penunjuk atau successor: satu untuk mengakses elemen berikutnya (Next) dan satu lagi untuk kembali ke elemen sebelumnya (Prev).

Untuk mengelola rangkaian data ini, terdapat dua penunjuk utama sebagai kendali list:

First: Pointer yang mengunci lokasi elemen paling awal di dalam list.
Last: Pointer yang mengunci lokasi elemen paling akhir di dalam list.

#### 1. Searching
Searching adalah salah satu operasi fundamental dalam manajemen list yang bertujuan untuk menemukan node tertentu di dalam rangkaian data. Secara teknis, proses ini bekerja dengan metode penelusuran satu per satu (traversal), di mana program mengunjungi setiap elemen secara berurutan mulai dari awal dan akan berhenti seketika saat data yang diinginkan telah ditemukan.

#### 2. Delete
Fungsi penghapusan pada Doubly Linked List terdiri dari deleteFirst untuk menghapus node pertama, deleteLast untuk menghapus node terakhir, dan deleteAfter untuk menghapus node yang berada di tengah linked list.

// DELETE
void deleteFirst(linkedlist &List){
    if(isEmpty(List) == true){
        cout << "List kosong! Tidak ada node yang dihapus." << endl;
        return;
    }
    address pDel = List.first;
    if(List.first == List.last){
        List.first = List.last = Nil;
    } else {
        List.first = pDel->next;
        List.first->prev = Nil;
        pDel->next = Nil;
    }
    pDel->prev = Nil;
    cout << "Node " << pDel->isidata.nama << " berhasil dihapus dari posisi pertama." << endl;
    dealokasi(pDel);
}

void deleteLast(linkedlist &List){
    if(isEmpty(List) == true){
        cout << "List kosong! Tidak ada node yang dihapus." << endl;
        return;
    }
    address pDel = List.last;
    if(List.first == List.last){
        List.first = List.last = Nil;
    } else {
        List.last = pDel->prev;
        List.last->next = Nil;
        pDel->prev = Nil;
    }
    pDel->next = Nil;
    cout << "Node " << pDel->isidata.nama << " berhasil dihapus dari posisi terakhir." << endl;
    dealokasi(pDel);
}

void deleteAfter(linkedlist &List, address Prev){
    if(isEmpty(List) == true){
        cout << "List kosong! Tidak ada node yang dihapus." << endl;
        return;
    }
    if(Prev == Nil || Prev->next == Nil){
        cout << "Node sebelumnya (Prev) tidak valid atau tidak ada node setelahnya!" << endl;
        return;
    }
    if(Prev->next == List.last){
        deleteLast(List);
        return;
    }
    address pDel = Prev->next;
    Prev->next = pDel->next;
    pDel->next->prev = Prev;
    pDel->next = Nil;
    pDel->prev = Nil;
    cout << "Node " << pDel->isidata.nama << " berhasil dihapus setelah node " << Prev->isidata.nama << "." << endl;
